# 🔮 マナ・ガーデン (Mana Garden) 〜箱庭魔法都市〜

剣と魔法の世界を舞台にした、明るく・かわいく・ポップな箱庭街づくりシミュレーションゲームです。

## 🌟 主な特徴

- **立体感のあるクォータービュー（アイソメトリック）グリッドマップ**
  - 8x8の斜め見下ろし型グリッド上に、ファンタジーならではの建物を自由に配置できます。
- **自律型キャラクターAI（冒険者 vs スライム）**
  - 街の境界線からスライム（魔物）が時折現れます。
  - 「酒場」や「兵舎」を建設すると、ちびキャラの冒険者（騎士・魔導士）たちが男女ペアで召喚されます。
  - 冒険者たちは自動で街を巡回し、スライムを発見すると自動戦闘！経験値を獲得してレベルアップします。
- **政治・勢力クエストシステム**
  - 「王宮」「冒険者ギルド」「大魔導書院」という3つの勢力が、街の発展に応じた要望（クエスト）を出してきます。
  - 要望を達成して友好度を高めることで、特殊なバフや魔法、新ユニットがアンロックされます。
- **Web Audio APIによるサウンドエンジン**
  - 起動すると優しくファンタジーなオリジナルBGM（アルペジオ進行）が自動でループ再生されます。
  - クエスト達成時には、複数の周波数を合成した透き通るようなFMシンセ風のクリアなベル音が流れます。
- **AI生成素材（NanoBanana）の自動透過処理**
  - ポップでカラフルなすべてのグラフィックアセットは、AI画像生成によって作成されています。
  - プログラム側でキャンバスを用いたリアルタイム透過処理（白色背景除去）を行い、自然にゲームへ統合しています。

## 🎮 遊び方

1. `index.html`をダブルクリックするか、ローカルサーバー（Live Serverや`npx serve`など）で起動します。
2. 起動すると、ロード画面の後に「ゲームスタート」ボタンが活性化します。
3. 画面下部の建築ショップから建物を選んで、グリッドに配置してください。
   - **民家**: ゴールドを生産し、住民最大枠を増やします。
   - **木こり小屋**: 木材を生産します。
   - **魔力塔**: 魔法に必要なマナを生産します。
   - **冒険者の酒場**: 冒険者（魔導士）を召喚します。
   - **騎士団兵舎**: 防衛用の騎士を召喚します。
4. マップをドラッグして移動（パン）、ホイールスクロールでズームが可能です。
5. 画面左側の魔法書（スペル）で「成長の雨」などの強力な魔法を唱えて街の成長を助けましょう。
6. 画面右側の勢力クエストの条件を満たしたら「報告する」ボタンを押し、友好度と資源を獲得しましょう。

## 🛠️ テクノロジー

- **フロントエンド**: HTML5, Vanilla CSS, Vanilla JavaScript (ES6 Modules/Procedural)
- **オーディオ合成**: Web Audio API (振幅エンベロープ、周波数変調シンセサイザー、オシレーターアルペジエーター)
- **画像透過処理**: Canvas API ImageDataピクセル処理
