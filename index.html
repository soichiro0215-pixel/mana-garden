1. 基本ルールとクリア目標

あなたは空中を漂う魔法の浮遊島の領主です。ゴールド、木材、マナを生産する施設を建て、街を豊かに育てましょう。右側の「王宮」「ギルド」「大魔導書院」の3大勢力から届くクエストを完了して友好度を高めることで、特別な支援（ゴールド生産バフ、精鋭ユニット、魔法割引）を獲得できます。

2. 冒険者とスライム (防衛)

街の周りからかわいいスライムが侵入してきます。放置すると住民の「幸福度」が下がり、ゴールドの生産性が低下します。
「酒場」や「兵舎」を建てることで、ちび騎士や魔導士が男女ペアで召喚されます。彼らは自動でパトロールし、スライムを発見すると戦って討伐します。討伐するとゴールドが手に入り、冒険者は経験値を得てレベルアップします！

3. 開拓と整地

島には最初に枯れ木 🪵、魔力岩 💎、雑草 🌿 などの障害物があります。クリックして少量の資源（ゴールド/マナ）を消費することで開拓でき、木材やゴールドを獲得できます。きれいに開拓して建物を建てましょう。

4. 魔法の使用

マナを消費して強力な魔法を唱えられます：
・成長の雨 🌧️: 一定時間、全資源の生産速度が2倍になります。
・マナの奔流 🌀: ゴールドをマナに瞬時に変換します。
・スライム召喚 💧: 冒険者に討伐させてEXP/資源を稼ぐためスライムを呼びます。
・浄化の光 ☀️: 選択したスライムや障害物を即座に消し去ります。

5. 操作方法

・画面のスクロール: マウスドラッグ、またはホイール回転（スワイプ）でマップを自由にスクロールできます。
・拡大縮小（ズーム）: 右上の「➕ / ➖」ボタン、または Ctrl + ホイール回転 で調整できます。
・UI折りたたみ: 左右パネルの「◀ / ▶」ボタンを押すとパネルを隠して、マップを広く見渡せます。
